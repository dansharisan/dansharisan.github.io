About Me

I'm Nghia Le, some friends call me Sten. I'm from Vietnam and currently residing in Chiba, Japan.
I've been working as a web developer since graduation (2012), and have accumulated a wide variety of skills ranging from server-side languages (PHP, Java, C#) to client-side languages (JavaScript, HTML, CSS) and system architecture.
My favorite technology stack is LEMP (Linux + Nginx + MySQL + PHP), mixing with some other useful libraries such as VueJS, Bootstrap... but not I'm not limiting myself to them.
I'm always open to learn new things as my final goal is to build maintainable, user-friendly software that solve problems and improve productivity in general.
